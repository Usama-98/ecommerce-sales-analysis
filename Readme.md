# E-Commerce Sales Dashboard

This project analyzes e-commerce sales data using SQL and Tableau.

## Tools Used
- SQL (SQLite)
- Tableau Public
- GitHub

## Dashboard Features
- Monthly Revenue Trend
- Revenue by Category
- Top Customers
- Interactive Filters

## SQL Concepts Used
- Joins
- Aggregations
- GROUP BY / HAVING
- CTEs
- Subqueries
- DENSE_RANK()

## Project Structure
- Data files
- SQL analysis queries
- Tableau dashboard
- Dashboard screenshots







Top 10 Customers by Total Spending

This analysis identifies the highest-value customers based on cumulative purchase amounts. The results can help businesses target loyalty programs, retention strategies, and personalized marketing campaigns toward their most profitable customers.



Monthly Revenue Analysis

This analysis tracks revenue trends over time by aggregating sales at the monthly level. The results help identify seasonality, growth patterns, and periods of strong or weak sales performance. The visualization can be used by management to monitor business performance and support forecasting decisions.


Revenue by Product Category

This analysis calculates revenue generated by each product category by combining product pricing information with sales quantities. The results help identify the most profitable categories and support strategic decisions related to inventory management, marketing investments, and product portfolio optimization.

Top Products by Revenue

This analysis identifies the highest-performing products based on revenue generated from sales. Revenue is calculated using product price multiplied by quantity sold. The results help businesses understand which products contribute the most to overall sales performance and profitability.
